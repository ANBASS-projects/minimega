What is minimega?

minimega is a tool for launching and managing virtual machines. It can run distributed across a cluster, or on a laptop. minimega is fast, easy to deploy, and can scale to run across massive clusters.

Getting Started

The Installation Guide contains instructions for fetching and installing minimega. Once you've installed it, the User's Guide will help you get started using minimega. It also provides online help for individual commands.
